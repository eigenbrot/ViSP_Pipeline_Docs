run_polarization_calibration¶

ViSP_Pipeline.ViSPPipeline.run_polarization_calibration(raw_pol_dir, raw_dark_dir, raw_lamp_gain_dir, raw_solar_gain_dir, num_bins_x=1, num_bins_y=1, dark_cal='DarkCal.fits', lamp_gain_cal='LampGainCal.fits', solar_gain_cal='SolarGainCal.fits', geometric_cal='GeoCal.fits', threads=1, gen_noise_frames=False, **unused_kwargs)[source]¶

Generate a set of reduced, macropixel’d ViSP data for use by the PACPipeline.

This function basically constructs the correct config and kwarg objects and then does all of the actual processing via calls to generate_intermediate_objects and ViSPPolCalPipeline.start_polcal_multiproc.

Parameters:

raw_pol_dir : str

Location of directory that contains raw Calibration Sequence (CS) data

raw_dark_dir : str

Location of the directory containing raw dark frames associated with the CS

raw_lamp_gain_dir : str

Location of the directory containing raw lamp gain frames associated with the CS

raw_solar_gain_dir : str

Location of the directory containing raw solar gain frames associated with the CS

num_bins_x : int

The final number of macropixels to extract/average from the processed CS data, spectral dimension.

num_bins_y : int

The final number of macropixels to extract/average from the processed CS data, spatial dimension.

dark_cal : str

Location of the processed DarkCalibration object to either load or save

lamp_gain_cal : str

Location of the processed LampGainCalibration object to either load or save

solar_gain_cal : str

Location of the processed SolarGainCalibration object to either load or save

geometric_cal : str

Location of the processed GeometricCalibration object to either load or save

threads : int

Maximum number of cpus/threads to use during reduction

gen_noise_frames : bool

If True then uncertainty estimates will be generated for the reduced CS step data

unused_kwargs : dict

Keyword arguments that exist in the config file, but are not used by this function
Returns:

str

The location of the directory associated with the reduction of the CS
