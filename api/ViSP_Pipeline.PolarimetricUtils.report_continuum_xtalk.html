report_continuum_xtalk¶

ViSP_Pipeline.PolarimetricUtils.report_continuum_xtalk(fitsData: ViSP_Pipeline.Data.FitsData, cont_lims: List[int])[source]¶

Determine the cross-talk from both Q and U to I in a continuum region and update PA&C database

For our purposes cross-talk is simply defined as the ratio of [QU] to I in a specified wavelength region. After this ratio has been computed the median is computed over all spatial positions.

Parameters
fitsDataData.FitsData

Object containing reduced ViSP Stokes data

cont_limslist

2-element list containing the high and low pixel limits of a continuum region in the provided data

Returns
float

Q/I cross-talk

float

U/I cross-talk
