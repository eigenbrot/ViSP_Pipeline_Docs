find_beam_angles¶

ViSP_Pipeline.SolarGainCalibration.find_beam_angles(beam1: numpy.ndarray, beam2: numpy.ndarray, numotsu: int = 5, numtheta: int = 1500, theta_min: float = - 0.03490658503988659, theta_max: float = 0.03490658503988659, debug: bool = False) → numpy.ndarray[source]¶

For both beams, find the angle between the slit hairlines and the spectral image axis

Spectral image axis refers to the pixel grid

Parameters
beam1numpy.ndarray

Beam1 data

beam2numpy.ndarray

Beam2 data, mirrored

numotsuint

The number of Otsu thresholding iterations to perform

numthetaint

The number of theta samples to use when constructing the Hough transform

Returns
numpy.ndarray

A length 2 array containing the hairline angle for each beam. They should probably be the same.
