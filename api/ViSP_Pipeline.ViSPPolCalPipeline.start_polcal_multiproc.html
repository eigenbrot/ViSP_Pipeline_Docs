start_polcal_multiproc¶

ViSP_Pipeline.ViSPPolCalPipeline.start_polcal_multiproc(config, pol_kwargs)[source]¶

Manage a set of PolCal reduction processes using multiprocessing.

Each step in the Calibration Sequence (i.e., separate file) is assigned its own process and then the processes are started in such a way that there are never more than the user-defined maximum running at a single time.

Parameters:

config : configparser.ConfigParser

The object containing user-defined reduction options

pol_kwargs : dict

The Polarimetric Calibration keyword arguments that contain the intermediate calibration objects and any other reduction options
Returns:

None
