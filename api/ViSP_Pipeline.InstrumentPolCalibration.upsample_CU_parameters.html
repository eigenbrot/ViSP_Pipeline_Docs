Take CU parameters fit to a subsample of the ViSP frame and upsample to some new sampling

Up-sampling is done using nearest-neighbor interpolation in such a way to ensure that each macropixel (i.e. bin) is equally represented in the upsampled data. The exception to this is when there is not an integer multiple between the current binning and desired binning.

Despite the 2D nature of ViSPâs detector, CU parameters are saved/loaded by the PA&C Modules with three dimensions. Thus, the shapes of the input and output will be (X, Y, L, Z), where L is the extra PAC dimension and Z is the number of parameters.
