get_matching_beams¶

ViSP_Pipeline.SolarGainCalibration.get_matching_beams(fitsData: ViSP_Pipeline.Data.FitsData, angles: numpy.ndarray, beam_offset: numpy.ndarray, state_offset: numpy.ndarray, errData: Optional[ViSP_Pipeline.Data.FitsData] = None, clean_data: bool = False) → Tuple[numpy.ndarray, numpy.ndarray, numpy.ndarray, numpy.ndarray][source]¶

Extract beam 1 and 2 and transform them so they share the same coordinates.

In other words beam_1[i,j] and beam_2[i,j] will correspond to the same _physical_ location (i.e., field position before entering ViSP optics). This is done by first mirroring beam 2 and then applying the appropriate rotations and shifts.

Each HDU (i.e., modulator state) in the input data will added to the third dimension of the output data.

Parameters
fitsDataData.FitsData

The distorted (raw) input spectra

anglesnumpy.ndarray

The hairline angles for the two beams

beam_offsetnumpy.ndarray

The (x,y) offsets that match the mirrored beam 2 to beam 1

state_offsetnumpy.ndarray

Shape (N,2,2) where N is the number of modulator states. The 2nd dimension corresponds to beam 1 or beam 2 and the last dimension contains the (c,y) shift needed to shift each modulator to a global mean.

errDataData.FitsData

Uncertainty assocaited with fitsData

Returns
numpy.ndarray

A 3D array containing the data of the un-distorted beam 1 at N modulator states

numpy.ndarray

A 3D array containing the data of the un-distorted beam 2 at N modulator states

numpy.ndarray or None

A 3D array containing the uncertainty for the un-distorted beam 1 at N modulator states

numpy.ndarray or None

A 3D array containing the uncertainty for the un-distorted beam 2 at N modulator states
