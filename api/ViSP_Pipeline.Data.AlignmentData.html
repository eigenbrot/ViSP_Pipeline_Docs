AlignmentData¶

class ViSP_Pipeline.Data.AlignmentData(data_dir=None, dataset_ID='', memmap=None)¶

Bases: ViSP_Pipeline.Data._TaskData

AlignmentData is an extension of FitData designed to only hold data from the Alignment InstrumentProgramTask (DKIST004). It inherits all of the functionality of FitsData and defines an extra method that loads data from a directory, provided it’s DKIST004 value is ‘Alignment’.

Initialize the empty object and maybe load some data.
