extract_M12_map¶

ViSP_Pipeline.PolarimetricUtils.extract_M12_map(M12dir: str)[source]¶

Aggregate a collection of individual Stokes data files into a single 4-dimensional map

The individual files should correspond to ViSP slit positions and should have all 4 Stokes parameters. The resulting file will have 4 ImageHDUs, corresponding to the 4 Stokes vectors. Each ImageHDU will have shape (X, S, W) where X is the original spatial dimension of the ViSP data, S is the number of slit positions, and W is the wavelength dimension.

The output file is written to M12dir/full_maps.fits

Parameters
M12dirstr

Location of directory containing individual files for each slit position. These files are very likely the result of save_M12_map_frame.
