DataSetData¶

class ViSP_Pipeline.Data.DataSetData(data_dir=None, dataset_ID='', suffix='.fits')[source]¶

Bases: object

DataSetData is designed to handle the aggregation of a bunch of files that belong to a single ViSP data set. A data set is generally the set of spectra of all spatial positions of the ViSP slit. This class reads all files in a directory and populates a dictionary that maps spatial position to filename. It can then create and return instances of the SlitPositionClass based on this dictionary. In this way the entire directory only needs to be read once, which saves a lot of time.

Initialize a few parameters and read the directory, if specified.

Parameters:

data_dir : str

Path to directory in which to look for data

dataset_ID : str

Only check data from this data set. A check is made against each file’s ID_004 header value. Can be a blank string to just load all data sets.

suffix : str

Optional file mask. The full mask is DATA_DIR/*SUFFIX. Defaults to “.fits”

Attributes Summary
