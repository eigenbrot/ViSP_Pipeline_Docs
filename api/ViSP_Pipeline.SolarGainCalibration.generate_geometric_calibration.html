generate_geometric_calibration¶

ViSP_Pipeline.SolarGainCalibration.generate_geometric_calibration(angles: numpy.ndarray, beam_offset: numpy.ndarray, state_offset: numpy.ndarray, spec_shifts: numpy.ndarray, output: str, primary_header: astropy.io.fits.Header, image_header: astropy.io.fits.Header, overwrite: bool = True) → ViSP_Pipeline.Data.FitsData[source]¶

Save the parametrization of the ViSP geometric distortions to a FitsData object.

Each of the 4 parameters are saved to a separate Data HDU and the Primary Header populated with DKIST metadata.

Parameters
anglesnumpy.ndarray

Shape (2,). The hairline angles for the two beams.

beam_offsetnumpy.ndarray

Shape (2,). The (x, y) offset between beam 2 and beam 1.

state_offsetnumpy.ndarray

Shape (M, 2, 2). The first dimension corresponds to M modulator states, the second dimension corresponds to the two beams, and the last dimension contains the (x, y) offsets between each modulator state and the mean location of all modulator states for each beam.

spec_shiftsnumpy.ndarray

Shape (2, X), where X is the number of pixels in the spatial dimension. For each of the 2 beams, the spectral shift at each of the X spatial pixels.

outputstr

Path when the Geometric Calibration object will be saved.

primary_headerastropy.io.fits.header.Header

The primary header of the data used to generate the Geometric Calibration. This header will be used to updated the Primary HDU of the resulting Geometric Calibration object.

image_headerastropy.io.fits.header.Header

A header from a data HDU used to generate the Geometric Calibration. The NAXIS[1|2] parameters are taken from this header for future verification.

overwritebool

If True, overwrite any previous data residing at the location specified by output.

Returns
Data.FitsData

A Geometric Calibration object. Each of the 4 geometric distortion parameters is saved to a separate data HDU and the headers attempt to be descriptive.
