telescope_pol_correction¶

ViSP_Pipeline.ScienceCalibration.telescope_pol_correction(fitsData, teleCal, errData=None)[source]¶

Apply an inverse Telescope Mueller matrix and update header values

The inverse TM captures all mirrors from M1 to M6 (inclusive)

Parameters:

fitsData : Data.FitsData

Data to apply the inverse TM to. Should have 4 data HDUs (stokes vectors).

teleCal : numpy.ndarray

A (4, 4) array containing the Mueller matrix for the inverse of mirrors 1 - 6

errData : Data.FitsData

Uncertainty estimates on fitsData

include_parallactic : bool

If True the returned Mueller matrix will include rotation by the parallactic angle and will thus put the data into a common solar reference frame
Returns:

Data.FitsData

The input data with the polarimetric contributions of M1 - M6 removed

Data.FitsData or None

If uncertainties are provided then this will contain the uncertainty associated with the previous output.
