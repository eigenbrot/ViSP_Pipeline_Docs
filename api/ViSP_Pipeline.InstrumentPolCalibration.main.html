Run raw data (i.e., a PolCal CS observation) through the Instrument Polarization Calibration pipeline

The steps in the pipeline are:

  1. Perform ViSP reductions on entire CS data set

  2. Fit CU parameters and generate demodulation matrices from the resulting SoCCs

So basically, this pipeline calls the normal ViSP pipeline and then uses PA&C Modules to further analyze the reduced PolCal data.
