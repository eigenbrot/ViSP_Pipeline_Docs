generate_noise_frame¶

ViSP_Pipeline.generic.generate_noise_frame(fitsData: ViSP_Pipeline.Data.FitsData) → ViSP_Pipeline.Data.FitsData[source]¶

Create a pixel-uncertainty estimate based on known detector properties

The Poisson (i.e., counting) noise associated with each pixel is (in electrons):

P = sqrt( DN / N / g )

where DN is the raw pixel count (in ADU), N is the number of accumulations per FPA, and g is the detector gain (in ADU per electron).

The total noise estimate is then just the quadrature sum of P and the detector read noise.

Parameters
fitsDataData.FitsData

The raw data to use to generate a noise estimate

Returns
Data.FitsData

The pixel uncertainties
