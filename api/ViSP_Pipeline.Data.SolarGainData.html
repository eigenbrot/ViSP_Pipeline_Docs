SolarGainData¶

class ViSP_Pipeline.Data.SolarGainData(data_dir=None, memmap=None, **kwargs)¶

Bases: ViSP_Pipeline.Data._TaskData

SolarGainData is an extension of FitData designed to only hold data associated with SolarGain. It inherits all of the functionality of FitsData and defines an extra method that loads all data from a directory who’s headers match the following:

Key Value — —–

DKIST004 Gain PAC__002 Clear

TELSCAN [‘Raster’, ‘Random’, ‘Spiral’]

Initialize the empty object and maybe load some data.
