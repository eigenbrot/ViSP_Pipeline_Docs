extract_map¶

ViSP_Pipeline.generic.extract_map(fitsData, x_start, num_x_avg, errData=None)[source]¶

Take a series of N (x,y) 2D images (spectra) and extract an (y,N) map by averaging along the spectral dimension. The user specifies the spectral channels over which to average each image.

If an uncertainty data set is provided the pixel uncertainty will be correctly propagated through the extraction process.

Parameters
fitsDataData.FitsData

The data to be extracted. Each HDU in fitsData.data_hdus will become a new row in the output image.

x_startint

The y (spectral) index that defines the start of the region to average

num_x_avgint

The number of spectral channels to be used in the average

errDataData.FitsData

Per-pixel uncertainty estimates corresponding to the data in fitsData

Returns
Data.FitsData

If N = len(fitsData.data_hdus) and each HDU’s data has shape (x,y) then this object will have a single data HDU whoes data has shape (x,N). The Primary HDU’s header is copied from the input data, and the data HDU will have a blank header.

Data.FitsData

If errData is provided then a second FitsData object is returned that contains the propagated pixel uncertainties. The Primary HDU’s header is copied from errData.
